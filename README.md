# 자기소개 인터뷰 수행평가 앱

2026학년도 신명중학교 1학년 영어 말하기 수행평가 (자기소개 인터뷰) 진행용 웹 앱.

## 사용법

1. [앱 열기](https://engccer.github.io/interview-2026/)
2. "수행평가 시작" 클릭
3. 자기소개 (20초) → 질문 3개 (각 15초) → 완료

## 평가 구조

- **자기소개** (20초): 이름 + 2가지 이상 정보
- **질문 응답** (각 15초): 3개 카테고리에서 무작위 1개씩
  - Favorites (좋아하는 것)
  - Important People (소중한 사람)
  - Daily Habits (일상 습관)

## 키보드 단축키

| 키 | 동작 |
|----|------|
| Enter / Space | 다음 단계 |
| R | 다시 듣기 |
| Esc | 종료 (시작 화면으로) |

## 기술

- 단일 HTML 파일 (빌드 불필요)
- ElevenLabs TTS 음성 (영어: JM, 한국어: Do Hyeon)
- Web Audio API 벨 소리
- ARIA 접근성 지원
